# Security Policy

## Reporting Issues

Please report security issues privately by opening a GitHub security advisory
for this repository if available. If advisories are not enabled, open a minimal
issue that asks for a private contact path and avoid including exploit details.

## Supported Versions

The default branch is the only supported version until tagged releases begin.

## Sensitive Data

This plugin can use local dietary preferences and meal history. Do not publish
personal profile files, generated meal history, Woolworths account data, cart
contents, credentials, addresses, payment information, or browser session
artifacts.
